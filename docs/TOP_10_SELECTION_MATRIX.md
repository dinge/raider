# Top 10 Selection Matrix
## Scoring Framework for Raider App Ideas

### Scoring Criteria (1-5 scale each)

1. **Market Size** (1-5)
   - 5: $100M+ TAM, thousands of potential customers
   - 3: $10M+ TAM, hundreds of potential customers
   - 1: <$1M TAM, niche market

2. **Willingness to Pay** (1-5)
   - 5: $2000+/month, high business value
   - 3: $500-2000/month, moderate value
   - 1: <$500/month, low margins

3. **Technical Feasibility** (1-5)
   - 5: Can build MVP in 1-2 weeks
   - 3: Requires 4-6 weeks
   - 1: Complex, 3+ months

4. **Competitive Advantage** (1-5)
   - 5: Few competitors, unique AI approach
   - 3: Some competitors, differentiation possible
   - 1: Crowded market, hard to differentiate

5. **Regulatory Complexity** (1-5)
   - 5: No regulations, easy to launch
   - 3: Some compliance, manageable
   - 1: Heavy regulation, slow go-to-market

6. **Vision AI Value** (1-5)
   - 5: Core value from vision capabilities
   - 3: Vision helps but not essential
   - 1: Minimal vision component

7. **Multi-Agent Value** (1-5)
   - 5: Complex workflow, clear agent orchestration value
   - 3: Some multi-step benefit
   - 1: Could be single LLM call
